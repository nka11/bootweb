bootweb
=======

Boot your web server

Everything needed for a quick application server bootstrap.
(requires mongodb but may be overridable soon)

Dependencies :

    apt-get install mongodb
    
    npm install

Run tests :
    ./node_modules/vows/bin/vows --spec tests/main.js
